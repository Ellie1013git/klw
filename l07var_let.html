변수에 대해서 알아보자

변수를 사용하는 이유

  1. 데이터를 재사용하기 위해서
  2. 메모리 절약 가능(변수가 참조하지 않는 데이터는 생성즉시 바로 삭제되기 때문)

표기법과 변수표기법

  • 낙타 표기법 : nameName 문자와 문자사이 결합을 대문자로 하는 방법 (추천1*****)
  • 소문자 스네이크 표기법 : name_name 문자와 문자 사이를 _ 언더스코어로 결합 (추천2. 파일,폴더명)
  • 파스칼규칙 : NameName 낙타표기법과 같은데 시작이 대문자 (비추. 타입, 대명사)
  • 대문자 스네이크 표기법 : Name_Name (비추. 상수표기)

사용하면 안되는 이름

  • 예약어(약속어)는 사용하면 안된다.: a+b, a-b, if, for, let, var...
  • 미리 선언된 객체 : window, console, document...
  • 수로 시작하면 안된다. : 1name, 01name
  • 알아보기 힘든 이름은 안된다.: x,y 보다는 명확한 이름이 있는 것이 좋다.

var 과 let의 차이점

  • 변수를 생성하는 명령어로 var은 전체 영역에 선언되고 let은 영역을 구분한다.
  • var a=10; == sindow.a=10; (window : 브라우저객체, 전체영영)
  • var로 선언한 변수는 전체 영역에서 사용되기 때문에 hoisting(끌어당김)이 일어난다.
  • var은 동일한 이름의 변수를 여러번 선언 가능하다.
  • let은 같은 영역에서는 한번만 변수 선언이 되고 선언된 변수는 해당 영역에서만 유효하다.
