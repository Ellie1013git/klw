배열과 문자열(string)

배열(타입명:Array) : 순서(index)가 있는 자료를 저장하기 위해 만들어진 자료

//"안" "녕" "하" "세" "요" "~"의 배열로 안녕하세요~를 입력

순서가 있는 자료 배열 Array

  • Array : 순서대로 데이털르 참조하는 자료형
  • 선언법1 : new Array()
  • 선언법2 : ["사과","딸기","배",1, true, new Date()...]
  • 특정 순서(index)의 자료를 참조하는 법 : arr[0~n]
  • 자료의 길이 : arr.length
  • 자료를 뒤쪽에 추가 : arr.push(data)
  • 특정위치에 자료를 추가하거나 수정하는 것 : arr[0~n]=data

순서가 있는 자료인데 문자로만 된 기본형 데이터 String

  • String : 문자로만 된 순서가 있느 데이터로 js에서만 기본형 데이터로 정의해 사용중
  • 선언법1 : let s="안녕"
  • 기본형 데이터는 랩퍼 클래스 String : 문자열의 처리를 도와주는 함수를 지원, 존재하지 않는 필드를 참조하도록 도와준다
  • 문자열은 자주 사용하기 때문에 js에서는 문자열을 기본형으로 취급한다.
  • Array와 비슷하기 때문에, 유사한 필드나 함수를 지원한다.
