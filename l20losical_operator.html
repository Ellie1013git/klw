논리연산자로 복잡한 제언문을 작성할 수 있게 하자

논리연산은 비트연산과 비슷하다

  • || Or : 논리의 합
    • true || true : true
    • false || true : true
    • true || false : true
    • false || false : false
    • **논리 합은 한개라도 true가 있으면 true
  • && And : 논리의 곱
    • true && true : true
    • false && true : false
    • true && false : false
    • false && false : false
    • **논리 곱은 한개라도 false가 있으면 false

논리연산과 중단 점(논리연산이 멈추는 지점)

js 엔진은 논리연산에서, Or논리인경우엔 참(true)이, And논리인 경우엔 거짓(false)이 나올거라는 확신이 있다면 다음을 하지 않는다.

논리연산의 우선순위

우선순위는 곱하기>더하기.(논리연산과 산술연산중에선 수학연산이 우선순위) a || b && c || b && d => a || (b && c) || (b && d) a && b || c && b || d => (a && b) || (c && b) || d
